class kNN.FeatureVector(idn=None, name=None)¶

Represents a vector containing the features of an object.

The vector represents a concrete or abstract object with features that can be represented as numerical values. The vector should be interpreted as living in a multidimensional vector space where each dimension represents a distinct feature.

Example:

likes = 1 dislikes = -1 vector = Vector(1, ‘Winnie’) vector[‘Honey’, likes] vector[‘Pork’, dislikes]

import_feature_space()¶

Load CSV file containing feature space and set as instance space. Also sets features list.

File must have the following format:

ID Number,Name,Feature1,Feature2, … id0,name0,value01,value02, … id1,name1,value11,value12, …

. . . . …

This is the format natively exported by Microsoft Excel 2013.
